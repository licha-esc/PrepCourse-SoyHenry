.miTexto {
	color: green;
}


Selector -> .miTexto
Propiedad -> color
Valor-> green


El selector será una referencia que nos permitirá seleccionar un elemento HTML. La propiedad dirá qué característica queremos cambiar de ese elemento. Por último, el valor dirá cuál será el estilo que tendrá esa propiedad.

clases: se utiliza cuando se quiere aplicar un conjunto de estilos a múltiples elementos en la página.

id: se usa cuando se quiere aplicar estilos a un elemento único en la página. No debería haber 2 elementos con el mismo id 


Medidas:
em: es una medida relativa al tamaño de fuente base del documento / página en el que estamos trabajando. El navegador de cada dispositivo establece el tamaño de fuente base. Entonces el em es una cantidad de veces ese tamaño base.
font-size: 3em; Esto indíca que en el dispositivo en que se esté visualizando un tamaño distinto nuestro documento siempre va a haber una relación de 3 veces el tamaño de la fuente base. También permite usar decimales: 2.8 em; etc

 
*No es del todo correcto asignarle estilos a etiquetas como header, main, article... en algunos casos si pero es mejor ponerlos en un div adentro del header, main, etc.
